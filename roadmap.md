## How to spend your 20 hours each week (simple, consistent)

You’ll follow this rhythm, changing the mix as you advance:

- **Weeks 1–8 (Foundations):** 8h Python + 8h Math + 4h practice/build
- **Weeks 9–21 (Core ML):** 8h ML + 6h Math + 6h build
- **Weeks 22–39 (Deep Learning + CV/NLP):** 10h DL + 4h Math + 6h build
- **Weeks 40–52 (GenAI + Production):** 8h GenAI/MLOps + 2h Math + 10h build

## Small but important “rules” to make this work

1. **Never watch-only.** Every week must end with a pushed repo commit (even small).
2. **Keep one main course at a time.** If you fall behind, pause “extras,” not the spine.
3. **Math is practice-heavy.** Khan Academy mastery > video binge. ([Khan Academy][11])
4. **Portfolio over certificates.** Certificates help; demos get you hired.
